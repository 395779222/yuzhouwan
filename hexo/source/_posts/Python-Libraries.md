###__胶水语言__
	胶水语言，能够把用其他语言制作的各种模块 ( 尤其是 C/C++ ) 很轻松地联结在一起

###__脚本语言__
	ABC 语言的一种继承
	缩短传统的 编写 - 编译 - 链接 - 运行 ( edit-compile-link-run ) 过程
